## What is this thing?

Basically, I needed a place to put all my school work so my lecturers can see it. Instead of handing in loose papers or sending a million emails, I built this website. It has:

- All 15 units I'm taking this year
- My grades for CATs, exams, labs, and projects
- Photos of my marked scripts (so you can see the feedback)
- Signed observation checklists from practicals
- A few videos showing me doing actual lab work
- Some certificates I picked up along the way

---

## How to use it

1. Open the website
2. Click on any unit card
3. A popup shows all my assessments for that unit
4. Click "View Script" or "View Checklist" to see the actual marked document
5. Check out the Evidence section for videos and certificates

That's it. Nothing complicated.
